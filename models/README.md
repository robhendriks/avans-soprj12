**ojt-models**

## Installation

```sh
$ npm install git+ssh://git@robhendriks.co:ojt/ojt-models.git
```

## Usage

```js
var User = require('ojt-models').User;
```

**Available models:**
* Action
* Client
* Communication
* Contact
* Event
* Schedule
* Team
* User
